¿QUE ES UNA BASE DE DATOS?

Es un conjunto de informacion, almacenados en un sisstema informatico que permite gestionar, recuperar, actualizar y analizar datos de forma eficiente.

¿PARA QUE SIRVE UNA BD?

almacenar, organizar, gestionar y recuperar grandes cantidades de información de forma segura y rápida.

¿QUE ES UN CAMPO?

unidad mínima de almacenamiento de datos en una base de datos o registro

¿QUE ES UN REGISTRO?

conjunto de datos estructurados sobre una entidad, o un archivo que almacena eventos de un sistema

¿QUE ES UN DATO?

representaciones simbólicas (numéricas, alfabéticas, algorítmicas, espaciales, etc.) de un hecho, suceso o entidad, sin sentido propio hasta ser procesados

¿QUE ES UN DIAGRAMA ENTIDAD RELACION?

herramienta visual que representa la estructura de una base de datos, mostrando entidades (objetos/conceptos), sus atributos (características) y cómo se interrelacionan entre sí
