# ¿que es el commit en GITHUB?
¡es el momento en el que terminas de cerrar una caja, le pones cinta y escribes una etiqueta de que hay dentro!

# ¿que es push y pull?
¡push envia tus cambios locales al repositorio remoto, es como "subir" tus cambios!
¡pull descarga cambios del proposito remoto y los fusiona con tus cambios locales.es como "bajar" cambios del remoto!

# ¿como hacer un if en javascript?

¡El condicional if en JavaScript ejecuta un bloque de código si una condición específica es verdadera (true). Se estructura con if (condición) { // código }, pudiendo añadir else para el caso falso o else if para evaluar múltiples condiciones, esencial para la lógica de control de flujo!

# ¿como leer una variable desde la terminal?

¡Para leer una variable desde la terminal en un contexto de GitHub, generalmente se utilizan comandos de shell (Bash/Zsh) o la CLI de GitHub (gh). Usa echo $NOMBRE_VARIABLE para ver variables de entorno locales, git config --get <variable> para configuraciones de Git, o gh secret list para secretos de GitHub Actions.!
